City Skaters

Project On-going

Unity & C# - Mobile Game Dev - Game Design - Systems Design - Gameplay Programming - Version Control - 2D Animations - Design Patterns.

City Skaters is a casual, arcade-style endless runner game designed for mobile platforms. It features a procedurally generated map where players can perform tricks and overcome obstacles. This project aims to create a skateboarding game that doesn't use the same awkward on-screen controls as most mobile skate games and to create an experience that anyone can enjoy.

Currently under development, I am refining mechanics and adding new elements to the game to polish it further. If you have read this far and would love to take the test and give me some feedback, do not hesitate to contact me!
