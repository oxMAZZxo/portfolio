City Skaters

Project On-going

Unity & C# - Mobile Game Dev - Game Design - Systems Design - Gameplay Programming - Version Control - 2D Animations - Design Patterns.

City Skaters is a casual, arcade-style endless runner designed for mobile platforms. Built with intuitive swipe-based controls, it lets players perform tricks, grind rails, and overcome urban obstacles on a procedurally generated map. The goal was to reimagine the mobile skateboarding experience—moving away from clunky virtual joysticks and toward smooth, responsive touch input that feels natural and fun. From procedural map generation to custom-built input and scoring systems, City Skaters is a technical and creative deep-dive into mobile gameplay design. The game also features scalable architecture and performance-optimised systems tailored for mobile hardware. This project is still growing, with ongoing refinements to gameplay mechanics and content. If you’ve made it this far and want to try the game—or share some feedback—I’d love to hear from you!
