Pong Clone

P5* & JavaScript.

In my first-year at my university in the Programming 3 module, I created a clone of the classic Pong game using P5* and the sprite namespace. This module focused on exploring math and physics concepts through P5*, and I developed the Pong clone to demonstrate my understanding. Although coding the game was initially challenging, as I was new to JavaScript, I became more fluent in the language over the course of several weeks.
