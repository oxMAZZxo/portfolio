Job Application Tracker

WPF - C# - SQL - Microsoft SQL Server - SQL Server Management Studio - Visualisation

The Job Application Tracker is a comprehensive and user-friendly application designed to help job seekers manage their job applications effectively. It utilizes an SQL Server database to securely store job application data.

The visualisation page displays key metrics, such as the total number of job applications submitted, responses received, and other relevant statistics, allowing users to gain insights into their job search progress. The data page presents all application information in a detailed and organized manner.

Users have the ability to add new job applications, update existing ones, and track various stages of the application process, including responses and interviews. Currently, the application is tailored for my personal use; I developed it to address my challenge of keeping track of all my applications and to create an organized system for managing my job search. The development of the app is on-going, and I will be adding features, based on my needs when using it.
