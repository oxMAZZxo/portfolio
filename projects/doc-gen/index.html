Documentation Generator v0.1

WPF - C# - XAML - MVVM - System Design - Asynchronous Programming.

Documentation Generator is a productivity-focused desktop application designed to streamline the process of creating technical documentation. Built to parse XML comments directly from C# source files, it automatically generates clean, structured PDF documents, saving developers time and reducing repetitive effort. The tool supports both individual file selection and entire directory scanning, making it adaptable for projects of any scale. The goal behind this project was to reimagine documentation workflows by leveraging self-documented code, transforming existing comments into professional documentation with minimal extra effort. From XML parsing to PDF generation, the app is a deep-dive into building scalable, production-ready systems. Development is ongoing, with new features and refinements being added regularly. The app currently only supports C#, however I am aiming to add support for other languages as well and will soon be publicly available to download.
