Pong Clone

P5* & JavaScript.

Using the sprite namespace, I cloned the classic pong game in p5js. This took place in the Programming 3 Module in the first year of my university course. In that module we explored maths & physics concepts, using p5js, and to show my understanding I developed this clone pong game. Coding the game was a little difficult since java-script is not a language I used, but after several weeks, I became more fluent in it.
