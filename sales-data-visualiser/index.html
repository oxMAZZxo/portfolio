Sales Data Visualiser

WinForms & C# - Systems Design - Desktop Development - Problem Solving

This is a personal project I worked on for a few weeks. It was inspired by my current job at a pub and was designed to help us compare/track employee and business performance. The software takes in sales data, which the employer keeps track of, and drink-waste data that we collect on a day-to-day basis from a CSV file and visualizes it, which helps staff members understand their performance better. Users can easily select default directories for performance data and wastage data, saving time and effort. Additionally, users are allowed to select multiple performance periods to track overall sales growth and employee performance, making the process comprehensive and user-friendly.
